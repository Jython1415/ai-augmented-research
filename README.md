# Papers

AI-augmented research papers. Each subdirectory is a self-contained study.

## Studies

- **[model-collapse/](model-collapse/)** — How Bluesky discourse about AI model collapse diverged from the science. Discourse analysis of 2,835 posts coded on six dimensions against a 21-paper literature timeline.
- **[model-collapse-v2/](model-collapse-v2/)** — Citation distortion in model collapse discourse. Tag-based distortion analysis of 539 citation units across 6 knowledge epochs, finding distortion rates quadruple from 13.8% to 49.4% as discourse matures.
